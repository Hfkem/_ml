# nn0.py 輕量級深度學習引擎解析

這份文件解析了 `nn0.py` 的核心機制。該模組展示了現代深度學習框架（如 PyTorch）最核心的魔法：**自動微分 (Autograd)** 與 **梯度下降優化 (Gradient Descent Optimization)**。

---

## 核心元件介紹

### 1. Value 類別：會記住歷史的數字
`Value` 是這個引擎的心臟。當我們將數字包裝成 `Value(0.5)` 時，它不再只是一個單純的數字，而是一個帶有狀態的節點。

* **前向傳播 (Forward)**：當你對 `Value` 進行運算（如 `a + b`）時，它會建立一個新的 `Value`，並透過 `_children` 屬性紀錄來源，從而建構出一張**計算圖 (Computational Graph)**。
* **反向傳播 (Backward)**：當呼叫 `loss.backward()` 時，引擎會沿著計算圖逆向回溯，利用**連鎖律 (Chain Rule)** 自動計算每個變數對最終損失（loss）的影響力，即 **梯度 (grad)**。



---

### 2. linear(x, w)：特徵轉換
執行矩陣與向量的乘法：
$$y = Wx$$
神經網路透過調整權重矩陣 $W$，學習如何將輸入數據（如感測器訊號）映射到正確的輸出特徵空間。

---

### 3. softmax(logits)：將數值轉化為機率
神經網路輸出的原始數值（Logits）範圍極大且不規律。`softmax` 函數能將其轉換為正規化的機率分佈：

$$P(y_i) = \frac{e^{z_i}}{\sum_{j} e^{z_j}}$$

> **技術細節**：`nn0.py` 在實作中特別減去了 `max_val`。這是一個數值穩定性技巧，能有效避免指數運算產生的**數值爆炸 (Overflow)**。

---

### 4. Adam：聰明的步伐更新者
如果梯度（grad）決定了下山的「方向」，優化器就決定了邁出的「步伐」。

* **Adam (Adaptive Moment Estimation)**：結合了動量（Momentum）與自適應學習率。
* **優勢**：它會計算梯度的移動平均，使模型訓練更平穩且快速，不會因為單一數據的雜訊而產生劇烈震盪。
